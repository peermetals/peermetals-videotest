Render Video

Endpoint: POST /api/render-video

Generate a video reel for a listing

Request Body:

{
  "listingData": {
    "id": "listing-id",
    "title": "Product Title",
    "description": "Product description",
    "images": ["image-url"],
    "user_id": "user-id",
    "tier1_category": "Gold",
    "condition": "Mint",
    "weight": 1,
    "purity": ".999",
    "year": "2024"
  }
}

Example cURL:

curl -X POST https://peermetals-videotest.vercel.app/api/render-video \
  -H "Content-Type: application/json" \
  -d '{"listingData": {...}}'

Features

  • 26-second Instagram Reels (9:16 format)
  • AI-generated descriptions with Google Gemini
  • Automated rendering with Remotion
  • 15-minute max duration (Fluid Compute)
  • 3GB memory allocation
